# DocGoo

DocGoo is a web application designed to connect patients with clinics, making appointment scheduling easy and efficient. It provides a user-friendly interface for both patients and clinic staff to manage appointments and clinic information.

## Features

- *Clinic and Doctor Listings:* Display clinic and doctor details, including availability and fees.
- *Appointment Booking:* Patients can browse clinics, book appointments, and manage their bookings.
- *User Authentication:* Secure login and registration for patients and clinic staff.

## Technologies Used

- *HTML:* For structure and content.
- *CSS:* For styling and layout.
- *JavaScript:* For interactive features and functionality.
